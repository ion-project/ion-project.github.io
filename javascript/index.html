Javascript

A parte interativa do IonDesign é divida em duas seções: IonFramework e IonSelector, elas são responsáveis por cumprir tarefas diferentes que se complementam no trabalho de assegurar a experiência do usuário. Abaixo é possível ver a lista de funções da API.

IonFramework

O IonFramework é o controle de todos os componentes do IonDesign, assim como o gestor de interações do usuário. Ele é responsável por garantir uma melhor experiência do usuário, a partir dele é possível, por exemplo, mostrar diálogos, alertas e avisos, de forma geral, deixar mais rica a experiência do usuário com sua aplicação.

IonSelector

IonSelector é a parte da API Ion para interação com o DOM (Document Object Model), em outras palavas, o documento Web. Através desta APi é possível criar, remover e editar os elementos HTML da página de uma maneira mais fácil.

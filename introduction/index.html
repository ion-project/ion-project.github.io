Introdução

Para iniciar seu projeto com o IonDesign você poderia começar baixando a última versão compactada do framework.

DOWNLOAD Version 0.12.1 - 263 KB

Após terminar o download descompacte o arquivo em algum diretório de sua preferência. Você irá perceber que o arquivos descompactos são um conjunto de códigos css e javascript, além das fontes. A sua estrutura de projeto irá se parecer com isso.

/Seu projeto
    /ion
        /fonts
            material-icons.eot
            material-icons.ijmap
            material-icons.ttf
            material-icons.woff
            material-icons.woff2
        ion.css
        ion.js
        ion.min.css
        ion.min.js

É recomendado utilizar os arquivos minificados (Que contém ".min" no nome) se você necessita de mais performance e um carregamento mais rápido em suas páginas. Outra dica é colocar os arquivos javascript no final de suas páginas.

Estrutura básica do HTML

Para testar o funcionamento do IonDesign você pode implementar o código abaixo em sua página web. Apenas certifique-se que todos seus arquivos estão sendo corretamente referênciados no código. 

<!DOCTYPE html>
<html>
    <head>
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />

        <link href="ion/ion.css" type="text/css" rel="stylesheet" />
    </head>
    <body>
    </body>
    <script src="ion/ion.js"></script>
</html>
